# snwlee/resume

이 레포는 현재 사용 중인 **공식 이력서 웹사이트**입니다.

- Live: https://snwlee.github.io/resume/
- Korean: `src/index.html`
- English: `src/index-en.html`

## 방향

이 이력서는 아래 포지셔닝을 중심으로 관리합니다.

- 엔터프라이즈 연동 경험을 가진 백엔드 엔지니어
- SSO / 인증 / 운영 안정성 경험 보유
- AI 업무 자동화와 개인 서비스 운영 경험까지 연결 가능한 개발자

## 로컬 실행

```bash
npm install
npm start
```

## 빌드

```bash
npm run build
```

빌드 결과물은 `docs/` 아래에 생성되며 GitHub Pages 배포에 사용합니다.

## 참고

기존 `portfolio`, `resume-nextjs` 레포는 더 이상 메인 이력서 레포로 사용하지 않습니다.
현재 기준의 최신 이력서와 문구는 이 레포를 기준으로 관리합니다.
